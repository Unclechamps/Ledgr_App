## What It Is
This is a collaborative project which aims to create an inventory and online sales app for smaller retailers. Intuitive UI. 
## What We Used
* JavaScript
* Node.js
* PostgreSQL
* Mustache.js
* Bcrypt.js
* MaterializeCSS
* Masonry
* Sequelize
* Express.js
* jQuery
